hexo博客主题更换

在终端进入hexo的themes目录下

localhost:themes Paranorman$

clone你要使用的主题文件到该目录下:

git clone https://github.com/klugjo/hexo-theme-clean-blog.git themes/clean-blog

找到在hexo目录下的_config.yml文件，用文本编辑器打开，修改theme主题。

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: clean-blog

重新部署到本地hexo s,查看效果。主题更换完毕，再根据主题的yml文件修改主题相关信息即可。
